How to merge two PDFs in Ubuntu 18.04

Hojjat

Install pdftk:

sudo add-apt-repository ppa:malteworld/ppa

sudo apt install pdftk

Then simply call it:

pdftk file1.pdf file2.pdf cat output merged_output.pdf
